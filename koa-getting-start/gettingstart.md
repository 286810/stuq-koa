# Getting Start

## 创建项目

[koa-generator](github.com/base-n/koa-generator)支持Koa1.x和2.x，安装后，可以分别使用`koa`和`koa2`分别创建。

### Koa 1.x

```shell
$ koa helloworld
```

### Koa 2.x

```shell
$ koa2 helloworld
```

### 安装依赖模块

```shell
$ cd  helloworld
$ npm install
$ npm start
```
### 启动，并在浏览器中查看

```shell
$ npm start
```

open http://127.0.0.1:3000
